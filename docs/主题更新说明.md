## 💡 设计理念

### "Free Tools in one box" 的含义

1. **All-in-one**: 强调工具集成在一个平台
2. **Box**: 呼应 "TooliBox" 品牌名称
3. **Free**: 保持免费的核心价值
4. **简洁**: 更短、更易记、更有品牌感

### 中文版 "一盒免费工具"

- **一盒**: 对应 "one box"
- **免费工具**: 直译 "free tools"
- **简洁**: 4个字，朗朗上口
- **品牌感**: 与 "TooliBox" 品牌契合
